Pokemon Match Adventure

web application

A Pokemon themed memory-match game!

Technologies Used:

  • HTML5
  • CSS3
  • JavaScript
  • jQuery

I really had a lot of fun creating this game. From a design perspective, I wanted to capture the feelings of wonder and excitement that come with discovery and adventure - my favorite part of Pokemon. I tried to make sure to put in some fun easter eggs as well! From the coding perspective, I really wanted to focus oh my HTML, CSS and JavaScript skills. I also wanted to focus on the mobile responsiveness so that I could practice working with media queries.
