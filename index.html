Pokemon Match Adventure

web application

A Pokemon themed memory-match game!

Technologies Used:

  • HTML5
  • CSS3
  • JavaScript
  • jQuery
  • GIMP

Features:

  • Four Difficulty Levels
    • Unique Easy & Challenge Modes!
  • Statistic Tracker
  • Dynamically Created Game Boards
  • Randomized Card Backs & Win Screens
  • Music & Sound Effects
    • Mutable In-Game
  • Different Easter-Eggs for Different Levels!

I really had a lot of fun creating this game!

From a design perspective, I wanted to capture the feelings of wonder and excitement that come with discovery and adventure - my favorite part of Pokemon! I tried to make sure to put in some fun easter eggs as well, I definitely recommend playing it with the sound on!

From a coding perspective, I really wanted to focus on my HTML, CSS and JavaScript skills, putting emphasis on the mobile responsiveness so that I could practice working with media queries. I had a lot of fun with the "hidden" CSS animation!
